# My SQL的基本操作（总结）

今天我来总结一下My SQL的基本操作。

# 一、下载与安装

windows版本MySQL下载地址： http://dev.mysql.com/downloads/mysql/   安装步骤我就不演示了，这网上资料很多，比我写的都好，你们可以搜索看看，如果找不到合适的，可以私聊我或者在下面评论，我看见都会帮你们解决。

# 二、启动与关闭MySQL服务

第一步：点击开始菜单，选择“运行”。 第二步：点击运行之后，在输入框里面输入“cmd”，点击确定。 第三步：在命令行输入“net start mysql”,按回车键，开启mysql服务。 第四步：在命令行输入“net stop mysql”,按回车键，停止mysql服务。

# 三、连接MySQL

  1. 在cmd命令中输入mysql -u root -p点击回车后输入密码，便可进入MySQL数据库。
  2. 在cmd命令中输入在mysql -u root -p的后面直接输入密码（例：123456是我的MySQL数据库密码mysql -u root -p123456）直接进入MySQL数据库。

其中：-u为数据库用户名，root是mysq|默认用户名; -p为密码。

# 四、常用的SQL语句

注意：mysq|数据库中的命令必须以分号结尾"; "

1、show databases;   // 显示你有的数据库,就是指根目录都有那些文件夹;

2、create database 数据库名;   // 创建数据库;

3、use 数据库名;  // 选择要操作的数据库;;

4、create table 数据表名(字段名 字段数据类型)   // 创建数据表

5、show tables;  // 显示所选数据库中的表;

6、insert into 表 (列名,列名...) values (值,值,值...)   // 插入数据

7、select * from 数据表   // 查询数据表

8、delete from 数据表  // 删除数据表

9、update 数据表 set name ＝ '小明'   // 修改数据表内容

10、输入 exit;回车  // 退出MySQL数据库



  这些功能是最常用的功能，我只是简单总结一下，MySQL的功能远不止这些，我也是学生，正在学习，有什么不懂的评论或私聊我，我们一起讨论。
